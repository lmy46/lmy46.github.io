認識輸入（Input）

在前一章中，我們已經學會如何用 System.out.println 將文字輸出到螢幕。
這一篇要進一步學習：如何接收使用者輸入，讓程式更有互動性。

Java 中常用的輸入工具是 Scanner 類別，它可以從 System.in 讀取鍵盤輸入。

小提醒：避免輸入錯誤

如果先用 nextInt() 後面馬上接 nextLine()，會發生跳過的情形。
解法：中間加一行 scanner.nextLine(); 清除換行符。
