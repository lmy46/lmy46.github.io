變數（Variable）

變數是用來儲存資料的命名空間，必須先宣告型別。
Java 中變數也有作用範圍（Scope）與存取權限（Access Level）的概念。

Global vs Local 變數

類型 宣告位置 作用範圍 是否需初始化 常見用途
區域變數 (Local) 方法或區塊內 只能在該區塊使用 必須初始化 計算用、短暫記錄
成員變數 (Global) 類別中、方法外 整個類別中可用 有預設值 物件屬性、常數

存取修飾子（Access Modifiers）

在 Java 中，變數與方法都可以設定不同的存取權限，決定外部是否可存取。

修飾子 說明 使用範圍
public 完全開放 任意類別皆可使用
protected 類別與子類別共享 包含 package 與繼承
(無修飾詞) 預設（default） 僅限同一個 package
private 完全封閉 僅限當前類別中使用

常數（Constant）

若變數值需固定不可變，則可使用 final。
而若常數需被整個專案共用，會使用 public static final 宣告於類別中。

資料型態（Data Types）

Java 資料型態分為兩大類：

類型 說明 範例
基本型別（Primitive） 儲存基本數值 int, double, boolean, char
參考型別（Reference） 儲存物件參考 String, 陣列, 自訂類別

小整理：基本型別對照表

型別 大小 範例
byte 1 byte byte b = 10;
short 2 bytes short s = 1000;
int 4 bytes int i = 100000;
long 8 bytes long l = 10000000000L;
float 4 bytes（小數） float f = 3.14f;
double 8 bytes（更高精度） double d = 3.14159;
char 2 bytes（Unicode） char c = 'A';
boolean 1 bit（true/false） boolean flag = true;
